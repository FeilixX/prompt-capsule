# 分享文案模板

发帖 / 转发卡带时套用。占位符用 {} 包裹。不要把 delete_token 放进公开文案。
下面以小红书笔记为例,微信群 / 朋友圈 / 推特同理,改改语气即可。

## 模板 A:分享一个提示词
标题:这个提示词我用了一个月,直接给你封成卡带了
正文:
{一句话说这个提示词干嘛的}。
点开链接就是全文,你复制去喂给自己的 AI 就行:
{view_url}
有效期到 {expires_at},想要的话尽快存。

## 模板 B:工作流 / 长指令
标题:{场景} 我的一整套指令,打包成一个链接
正文:
省得你一句句抄。这是我在用的完整版:
{view_url}
直接甩给 agent,让它照着做。

## 模板 C:只发编码(小红书防降权)
小红书等平台发完整链接容易被限流 / 降权,所以这版正文**不放任何链接**,只发编码。用创建返回里的 `code_share_text`,或按下面套:
标题:{一句话卖点},编码拿走自己跑
正文:
{一句话说这个提示词干嘛的}。
提示词卡带编码:{code}
用法:跟你的 AI(装了 prompt-tape skill / MCP)说一句「读取提示词卡带 {code} 并执行」就行,它会自己取回正文照着做。
还没装的,主页搜 prompt-tape skill;{expires_at} 前有效,过期自动失效。
(编码是公开分享用的;delete_token 千万别发出去。)
